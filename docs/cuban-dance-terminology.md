# Cuban Dance Terminology Glossary

This glossary provides definitions for Cuban salsa (Casino) terminology used in SaborConFlow Dance classes.

## Basic Moves (Pasos Básicos)

### Footwork Fundamentals

| Term | Description |
|------|-------------|
| **Mambo** | Basic forward-and-back step pattern, foundational to salsa timing |
| **Bamboleo** | Side-to-side rocking motion, often used for styling |
| **Cross-open-cross** | Footwork pattern crossing in front, opening, then crossing again |
| **Cha-cha-cha** | Triple step pattern (syncopation) inserted into the basic |
| **Toe-heel** | Styling technique touching toe then dropping to heel |
| **Paso de son** | Traditional son step, precursor to modern salsa footwork |

### Weight & Movement

| Term | Description |
|------|-------------|
| **Weight shift** | Transferring body weight from one foot to another |
| **Splitting the basic** | When 123 movement differs from 567 (asymmetrical) |
| **Compound move** | Combining elements (e.g., mambo with travel) |

## Casino (Partner) Moves

### Essential Moves

| Term | Description |
|------|-------------|
| **Dile que no** | "Tell her no" - basic cross-body lead in casino |
| **Guapea** | Basic casino position and step, partners facing each other |
| **Enchufela** | Lead turns follow under arm, ending in closed position |
| **Sombrero** | "Hat" move - arm goes over leader's head |
| **Exhibela** | "Show her off" - presenting partner to audience/room |
| **Vacilala** | Follow turns in place while lead watches/admires |
| **Paseala** | "Walk her" - promenading partner around |
| **Caminala** | "Walk with her" - traveling together in partner hold |

### Setenta Family

| Term | Description |
|------|-------------|
| **Setenta** | "Seventy" - classic casino combination with multiple turns |
| **Setenta complicado** | Complex variation of setenta |
| **Setenta por arriba** | Setenta executed with arms overhead |

### Walking & Traveling

| Term | Description |
|------|-------------|
| **Dile que si** | "Tell her yes" - moving together in same direction |
| **Engaños** | "Tricks/deceptions" - faking direction changes |
| **Camina** | Walking step, traveling with partner |

### Turns & Rotations

| Term | Description |
|------|-------------|
| **Right turn** | Clockwise rotation |
| **Left turn** | Counter-clockwise rotation |
| **Pivot** | Rotation on the ball of one foot |
| **Spotting** | Focusing eyes on fixed point during turns for balance |
| **Clock drill** | Practice turning through all clock positions |

## Afro-Cuban Elements

### Rhythms & Dances

| Term | Description |
|------|-------------|
| **Yuka** | Afro-Cuban drum rhythm and associated dance |
| **Makuta** | Afro-Cuban ceremonial rhythm from Congo influence |
| **Ellegua** | Movement honoring the orisha of crossroads |
| **Pilon** | Grinding motion mimicking sugar cane processing |
| **Rumba** | Family of Afro-Cuban secular dances (guaguancó, yambú, columbia) |
| **Guaguancó** | Most popular rumba style, involves pursuit/avoidance |

### Clave & Rhythm

| Term | Description |
|------|-------------|
| **Clave** | Foundational 5-beat rhythmic pattern of Cuban music |
| **Clave guaguancó** | Specific clave pattern used in rumba guaguancó |
| **Son clave** | Traditional clave pattern (3-2 or 2-3) |

## Timing Concepts

| Term | Description |
|------|-------------|
| **Contratiempo** | "Against time" - dancing on the off-beats (2 and 6) |
| **On1** | Dancing with break step on beat 1 (common in LA style) |
| **On2** | Dancing with break step on beat 2 (NY mambo style) |
| **A tiempo** | "On time" - dancing on the main beats |

## Technical Terms

| Term | Description |
|------|-------------|
| **Lead** | The person initiating/directing movements (traditionally man) |
| **Follow** | The person responding to lead's direction (traditionally woman) |
| **Frame** | Arm/body position maintaining connection with partner |
| **Connection** | Physical and energetic link between partners |
| **Musicality** | Dancing in harmony with the music's structure and feeling |

## Style Elements

| Term | Description |
|------|-------------|
| **Pachanga** | Energetic, playful style with distinctive footwork |
| **Rumba transitions** | Moments where salsa dancing incorporates rumba elements |
| **Sabor** | "Flavor" - the personal style and feeling in dance |
| **Flow** | Smooth, continuous quality of movement |

## Class-Specific Terms

| Term | Description |
|------|-------------|
| **Pasos Básicos (PB)** | Beginner/fundamentals class |
| **Casino Royale (CR)** | Advanced partner work class |
| **Choreo Team** | Performance preparation group |
| **Eight count** | Standard phrase of dance (8 beats, 2 measures) |
| **Combination** | Sequence of moves linked together |
